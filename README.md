# Join the Xopo
Join the Xopo is a decentralized platform for artists to showcase and sell digital assets as non-fungible tokens. With intuitive navigation, and blockchain technology, Join The Xopo offers a secure and engaging experience for both artists and collectors. Explore the world of NFTs and unlock the potential of digital art with xopo. We're crafting a worldwide platform for creatives of all kinds, with NFTs as our canvas!

## Features
Secure and decentralized platform for artists to showcase and sell digital assets as NFTs
Intuitive navigation for a seamless user experience
Blockchain technology ensures authenticity and provenance of digital assets
System of smart contracts manages and powers the platform
Worldwide platform for creatives of all kinds, with NFTs as the canvas
Feel free to add more features or adjust as needed. Good luck with Join the Xopo!


## Technology Stack
'Join the Xopo' is built using Quasar, a Vue-based framework, and bundled with Webpack. The smart contracts that power the platform are written in Solidity and managed through a system of smart contracts.

## Usage
To use the platform (mint, buy, sell NFTs) or apply as an artist, the user will need a wallet (such as MetaMask) to connect to the platform and sign transactions. Here is a link to download MetaMask: https://metamask.io/download.html

## Install the dependencies
```bash
npm install
```
### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```

### Lint the files
```bash
npm run lint
```

### Build the app for production
```bash
quasar build
```

### Run jest unit tests
```bash
quasar test --unit jest
```
